# Folder Manager (fm) - golang-version

- 배포 및 사용 편의성을 극대화 하기 위하여 정적 컴파일러 언어인 Golang으로 포팅 된 버전입니다. Python 버전에 비해 안정성이 떨어 질 수 있으니 백업을 권장 드리긴 합니다.
- This version has been ported to Golang, a statically compiled language, to maximize deployment and usability. It may be less stable than the Python version, so we recommend backing up your copy.
